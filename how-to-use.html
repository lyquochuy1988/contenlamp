Content Lamp is very easy to use. You will learn how to use it in the next 10 minutes. Read on..

Before you start, here are the things you need:
  1. A folder that contains background images. Just to get you started rightaway we are providing a 'background_images' folder which has 10 images, which you can use royalty free. Needless to say, you can simply use images from your own camera!  You can also buy our 'Creator Graphics 500' set of 500 beautiful scenic images or our "Creator Graphics 1000" set of 1000 scenic images that you can use.

  2. A folder that contains foreground images, you can use flags or animals or people or clipart (anything you have legal permission to use)  These are images that will be mixed into the backgroud images to generate unique images.

  3. A text file that has slogans or witty sayings or things that famous people (including you) have said, which is added as text into the images. We are providing a file for you to use, but you can use any slogans you like. Just write them into a text file and load them up.

  4. A logo for the product you are trying to get publicity for.

  5. Make up your mind as to which platform you are needing to create images for, facebook, twitter , instagram, pinterest etc have varying requirements.

  6. If you want the generated images to have SEO friendly names, prepare a text file with a big list of your long tail keywords. Image marketing is powerful, but you still need to name the images correctly to get the most traction from your campaigns.
